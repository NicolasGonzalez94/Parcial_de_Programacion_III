# Food Store 🍕🍔

Aplicación frontend de una tienda de comida online desarrollada como parcial de Programación III.

## Descripción

Food Store es una tienda online que permite visualizar un catálogo de productos organizados por categorías, buscar productos por nombre, filtrar por categoría y agregar productos a un carrito de compras con persistencia en localStorage.

## Tecnologías utilizadas

- HTML5
- CSS3
- TypeScript
- Vite

## Funcionalidades

- Catálogo de productos organizado por categorías generado dinámicamente
- Búsqueda de productos por nombre
- Filtrado por categoría desde el nav
- Carrito de compras con persistencia en localStorage
- Visualización de nombre, precio, cantidad y subtotal por producto
- Cálculo del total de la compra
- Opción para vaciar el carrito

## Instrucciones para ejecutarlo

1. Clonar el repositorio
\```bash
git clone https://github.com/NicolasGonzalez94/Parcial_de_Programacion_III.git
\```

2. Entrar a la carpeta
\```bash
cd Parcial_de_Programacion_III
\```

3. Instalar dependencias
\```bash
pnpm install
\```

4. Levantar el servidor de desarrollo
\```bash
pnpm dev
\```

5. Abrir en el navegador
\```
http://localhost:5173/src/pages/store/home/home.html
\```
